# web3-shh

This is a sub package of [web3.js][repo]

This is the whisper v5 package.   
Please read the [documentation][docs] for more.

## Installation

```bash
npm install web3-shh
```

## Usage

```js
import {Shh} from 'web3-shh';

const shh = new Shh(
    'http://127.0.0.1:4546',
    options
);
```
